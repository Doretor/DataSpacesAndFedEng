Answers to questions:

1. Which components store the actual data?
providers/satellite_A/
providers/satellite_B/
providers/ground_station/

2. Which component enables dataset discovery?
metadata_catalog/

3. What advantages does the federated architecture provide?
Datasets can be discovered without centralizing the data, and datasets remain stored at independent providers. 
We only need a federated catalog where metadata is shared.

4. What limitations might such a system have?
All databases must use standardized data access interfaces.
